# PERSONAL WEBSITE / PORTFOLIO
## One-Page Case Study Site — Optional but Differentiating

---

### WHY

A one-page personal site:
1. Shows you're serious (most director-level candidates don't have one)
2. Proves your capability without industry context
3. Gives recruiters a place to see your story outside of LinkedIn
4. Can be built in 2 hours with Carrd or Notion

---

### WHAT TO INCLUDE

1. **Headline:** Same as LinkedIn
2. **Bio:** Same as LinkedIn About section (2-3 paragraphs)
3. **3 Case Studies:**
   - Scaling (3→120): What you built, how you did it, metrics
   - Revenue Ops ($0→$4M): Systems designed, results
   - Exit ($17M): Due diligence, integration, outcome

4. **Downloadable Resume:** Clean PDF
5. **Contact:** Email form (not plain text email — prevents spam)
6. **Testimonials:** 1-2 pull quotes from colleagues (optional)

---

### WHERE TO BUILD

**Option 1: Carrd.co (recommended)**
- $19/year for custom domain
- Simple one-page builder
- No coding required
- 30 minutes to set up

**Option 2: Notion**
- Free
- Use a public Notion page
- 1 hour to set up
- Can embed resume PDF

**Option 3: GitHub Pages**
- Free
- Requires basic HTML
- 2 hours to set up

---
