### STRATEGY: MAXIMUM PAY PER ROLE TYPE

```
TUTORING (private/platform)   → Target $30-60/hr   → DO DILIGENCE: check Tutriva/Wyzant rates
TEACHING ASSISTANT            → Target $30-45/hr   → Check SFU/KPU union agreements
ADMIN ASSISTANT               → Target $22-28/hr   → Check Robert Half ranges
RECEPTION/FRONT DESK          → Target $20-25/hr   → Check Indeed listings
LIBRARY ASSISTANT             → Target $22-27/hr   → Check VPL/BPL union rates
RETAIL/CUSTOMER SERVICE       → Target $20-23/hr   → Check Metrotown specific
CAMPUS/STUDENT SERVICES       → Target $23-28/hr   → Check SFSS/KUSA postings
```

**Due diligence protocol for every job:**
1. Look up the role on Indeed → note the range for Burnaby/Vancouver
2. Check Glassdoor for the specific company
3. Check PayScale / Robert Half for the role type
4. If tutoring: check Tutriva + Wyzant + TTS rates
5. Target the **upper quartile** of the range for that specific role type

**Key insight:** Pay varies wildly by role type. A tutoring centre pays $21/hr. Private tutoring pays $40-60/hr. Same "tutor" title, 2-3x difference. Always know the market before applying.
