# PDF RESUME GUIDE & OFFICIAL FORMAT
## Specs Every Resume Must Follow

---

### TECHNICAL SPECS

| Element | Spec |
|---------|------|
| Format | PDF/A (preferred) or DOCX (for Workday/Taleo) — never scanned image |
| File size | <500 KB (PDF) or <1 MB (DOCX) |
| Font | Arial or Calibri ONLY — 11pt body, 14-16pt name |
| Margins | 0.75" - 1" all sides (0.5" minimum) |
| Length | 2 pages maximum |
| Text | Selectable (not an image) — copy-paste must work |
| Metadata | Stripped (no author name, no company name in file properties) |
| File name | `CompanyName_RoleTitle_ARYAN_RESUME.pdf` (e.g., `EviSmart_HeadStrategicOps_ARYAN_RESUME.pdf`) |
| Line spacing | 1.0-1.15 |
| Bullets | Standard • or - only (no unicode, emoji, decorative) |
| Headers/footers | NEVER put content here — contact info in body top 3 lines |

**⚠️ IMPORTANT: See 32_ATS_TECH_SPEC.md for PER-PLATFORM rules. Each ATS has different requirements for DOCX vs PDF, font handling, date formats, and parsing quirks. Always cross-reference before generating.**

### CREATION PROCESS

**DO NOT:**
- Use Canva (creates image-based PDFs that ATS can't read)
- Use Photoshop, Illustrator, or design tools
- Use tables or columns (ATS reads left-to-right, top-to-bottom)
- Use headers/footers with text (ATS sometimes skips them)
- Use graphics, icons, or charts
- Use non-standard fonts

**DO:**
- Use Google Docs or Word
- Use single-column layout
- Export as PDF → verify text is selectable
- Test by copy-pasting entire resume into Notepad — if it's clean and readable, ATS will parse it

### VERIFICATION CHECKLIST (Before Sending)

- [ ] File is PDF/A, <500KB
- [ ] Text is selectable (copy-paste test passed)
- [ ] No tables, columns, graphics, icons
- [ ] No headers/footers with text
- [ ] Single-column layout
- [ ] Keywords from JD embedded naturally (2-4% density)
- [ ] Company name, role, dates all present
- [ ] File name matches format: `Company_Role_NAME.pdf`
- [ ] Metadata stripped
- [ ] Scanned for typos (read it aloud backwards)
- [ ] Scanned for personal info accuracy (phone, email, LinkedIn)
