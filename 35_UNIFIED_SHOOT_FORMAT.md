# UNIFIED MASTER OUTPUT — Every JD → Complete Battle Plan
## Core Kernel: After any JD, I output THIS. All sections. Every time.

---

### STRUCTURE (13 Sections)

```
1. HEADER          → Company, Role, Salary, Fit Score, NOC, TEER, PR
2. DNA EXTRACTION  → Language Registry, Value System, Cultural Cues, Anti-Patterns
3. ATS SPECS       → Platform, Font, Size, Format, Margins, Rules
4. DEVICE MAP      → Hardware they issue, OS, peripherals, IT policy
5. FRAMING GUIDE   → What to say, what NOT to say, agnostic rules
6. RESUME          → Plain text for proofreading + LaTeX source
7. COVER LETTER    → Plain text + LaTeX source
8. LINKEDIN        → 4-tier outreach messages
9. INTERVIEW       → 60-sec pitch, 5 Q&A, 3 STAR stories, keywords
10. CHECKLIST      → Pre-submission verification
11. FOLLOW-UP      → T+0 through T+21 cadence
12. FINOPS         → Salary breakdown, net pay, timeline
13. APPLICATION    → Direct link, submit instructions
```

---

### COMMAND UPDATE

| Command | Action |
|---------|--------|
| `SHOOT [company]` or `SHOOT [paste JD]` | Full master output: 13-section battle plan with LaTeX source |
| `SHOOT --fast [company]` | Quick output: resume + cover + link only |
