### DERIVED METRICS (Calculate Weekly)

| Metric | Formula | Healthy Range |
|--------|---------|---------------|
| Application volume | Total apps / 5 days | 12-15/week |
| Response rate | Responses / Apps sent × 100 | 10-20% |
| Interview conversion | Interviews booked / Apps sent × 100 | 5-10% |
| Stage advancement | Next round / Total interviews × 100 | 40-60% |
| Offer rate | Offers / Final rounds × 100 | 25-50% |
| Time-to-response | Days from apply to first response | 3-10 days |
| Time-to-offer | Days from apply to offer | 21-45 days |

---

### DASHBOARD INTERPRETATION

**If response rate < 5%:**
- Resume isn't resonating → rewrite using more JD keywords
- You're applying too late → prioritize fresh postings (<24 hours)
- Wrong roles → check if you meet basic requirements

**If interview conversion < 5%:**
- Application quality is weak → spend more time on Phase 4 (Alchemize)
- Cover letter isn't compelling → use templates from 12_COVER_LETTERS.md
- LinkedIn profile doesn't match resume → update per 08_OMNI_LINKEDIN.md

**If stage advancement < 40%:**
- Interview performance needs work → revisit 05_INTERVIEW_ALCHEMY.md
- You're not speaking their language → research company values deeper
- Technical gaps emerging → address with 21_SKILLS_GAP.md
